# Edward-Scissorhands
---

## 基于DragonEngine-骨骼动画片段切割器

利用DragonEngine开发的动画片段切割器，对模型文件的动画信息进行切割并导出为XML文件
DragonEngine读取模型的同时便可读取动画片段信息，进行动画状态机的控制
